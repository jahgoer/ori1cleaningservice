# Ori1 Cleaning Service Website

A fast, mobile-first static marketing website for Ori1 Cleaning Service, serving Birmingham, Coventry and the wider West Midlands.

## Technology

- Semantic HTML5
- Modern responsive CSS with Grid, Flexbox and lightweight animation
- Vanilla JavaScript for navigation, scroll reveals, FAQs and form feedback
- Inline/local SVG artwork for fast, dependable visual assets
- Netlify configuration for static publishing, caching and security headers

## Pages

The site includes Home, About, Services, Pricing, Service Areas, Gallery and Contact pages, plus a sitemap and robots file.

## Run locally

No build step is required. Open `index.html` directly in a browser or serve the directory with any static file server.

For Netlify, the publish directory is the project root as configured in `netlify.toml`.
